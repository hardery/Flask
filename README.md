### git
- git init
    - 在本地自己创建一个仓库 
- git clone xxx
    - 从指定位置获取克隆已有仓库
    
### git操作
- git pull
    - 从远端服务器拉取代码文件
- git push
    - 将本地代码文件推送到服务器
    
- git add
    - 添加版本追踪
    - 添加文件变更标记  
- git commit
    - 将变更提交到本地仓库
- git status
    - 查看工作空间状态
- .gitignore
    - git忽略规则
    - 写入.gitignore文件中的文件或目录，git将不会进行版本跟踪
